# SVG Icon Library

以下是各个图标的具体作用及其在业务场景中的使用分布说明：

## 1. 房态与主页导航类
- **home.svg**: 房源主页标识、底部导航栏“房源”入口
- **user.svg**: 租赁管理（标示租客状态）、专属经纪人入口、底部导航栏“我的”入口
- **clock.svg**: 历史记录时间标识
- **history.svg**: 开门记录及其他历史流水的入口
- **chevron-left.svg / chevron-right.svg**: 页面切换返回、列表项展开与确认操作
- **plus.svg**: 添加房源、新增门锁（买门锁入口）
- **more-horizontal.svg**: 列表页更多操作选项

## 2. 智能设备与控制类
- **lock.svg**: 智能门锁设备在线状态展示
- **unlock.svg**: 一键蓝牙开门操作
- **battery.svg**: 设备电量充足标识
- **battery-warning.svg**: 设备极低电量（低于 20% 时触发）警告标识
- **key-round.svg**: 基础密码管理、钥匙权限操作
- **fingerprint.svg**: 智能锁指纹录入入口
- **credit-card.svg**: 实体门卡添加与管理

## 3. 安全预警与异常处理类
- **alert-circle.svg**: 紧急开门告警状态、强提醒信息
- **shield.svg**: 隐私声明、整体资产安全防护标识
- **shield-check.svg**: 数据安全认证机制、底部导航大厅入口、已认证状态
- **shield-alert.svg**: 系统高危警告、更换手机号等安全验证步骤

## 4. 租务工单与增值服务类
- **wrench.svg**: 提交门锁报修、服务大厅状态标识
- **clipboard-list.svg**: 维修单/安装单等工单记录日志
- **phone.svg**: 拨打承租人电话、一键呼叫经纪人联系方式
- **bell-ring.svg**: 发起一键催租响铃报警
- **truck.svg**: 智能锁上门安装配送状态
- **shopping-cart.svg**: 在线购买设备（购物车选购）

## 5. 消息状态与通用界面操作类
- **check-circle-2.svg**: 成功状态（如验证通过、操作成功 Toast 提示）
- **check.svg / check-square.svg**: 单选与多选确认框
- **message-circle.svg**: 底部导航“消息”入口、微信分享功能
- **bell.svg**: 常规通知或消息聚合
- **info.svg**: 消息详情提示或状态信息概览
- **x.svg**: 模态弹框（Modal）关闭按钮
- **search.svg**: 搜索框及内容查找功能标识

## 6. 其他常用工具与推广拓展类
- **camera.svg / image.svg**: 添加图片、图库读取与朋友圈海报生成
- **mic.svg**: 语音录制或语音交互拓展
- **map-pin.svg**: 定位获取及门锁真实安装详细地址标记
- **share-2.svg**: 通用分享外链操作
- **qr-code.svg**: 二维码名片及推广分享
- **download.svg**: 保存海报或记录等文件下载
- **wallet.svg**: 个人中心账户钱包与提现管理
- **trending-up.svg**: 平台动态、收益趋势展示
- **help-circle.svg**: 帮助服务以及用户指南入口
- **user-cog.svg**: 专属经纪人资料展示及协助日志入口
- **edit-3.svg**: 编辑或重设智能门锁常驻密码
- **settings.svg / log-out.svg**: 用户系统基础设置及注销退出登录功能
- **sparkles.svg**: 推荐奖励高亮入口标识
- **heart.svg**: 系统收藏或状态点亮功能
